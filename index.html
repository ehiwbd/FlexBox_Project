Flexbox

Что такое Flexbox?

Flexbox (Flexible Box Layout) — это модуль макета CSS, разработанный для облегчения создания гибких, отзывчивых макетов без использования плавающих элементов или позиционирования. Он особенно полезен для компонентов пользовательского интерфейса и небольших макетов.

Flexbox состоит из двух основных компонентов:

  • Flex-контейнер — родительский элемент, к которому применяется свойство display: flex
  • Flex-элементы — дочерние элементы, размещенные внутри flex-контейнера

Основы Flexbox: контейнеры и элементы

Чтобы использовать Flexbox, нужно сначала создать flex-контейнер, применив к элементу свойство display: flex или display: inline-flex.

Свойства flex-контейнера:

  • display: определяет элемент как flex-контейнер
  • flex-direction: определяет направление flex-элементов
  • flex-wrap: определяет, должны ли flex-элементы переноситься
  • justify-content: выравнивает flex-элементы по главной оси
  • align-items: выравнивает flex-элементы по поперечной оси
  • align-content: выравнивает строки flex-элементов

Свойства flex-элементов:

  • flex-grow: определяет, как элемент может расти
  • flex-shrink: определяет, как элемент может уменьшаться
  • flex-basis: задает начальный размер элемента
  • flex: сокращенная запись для flex-grow, flex-shrink и flex-basis
  • align-self: переопределяет выравнивание для конкретного элемента
  • order: изменяет порядок элементов

Свойство flex-direction

Свойство flex-direction определяет направление, в котором flex-элементы размещаются в flex-контейнере. Оно устанавливает главную ось контейнера.

Интерактивный пример flex-direction:

3

Текущее значение: row

Возможные значения:

  • row (по умолчанию): слева направо
  • row-reverse: справа налево
  • column: сверху вниз
  • column-reverse: снизу вверх

Свойство justify-content

Свойство justify-content определяет, как браузер распределяет пространство между и вокруг flex-элементов вдоль главной оси контейнера.

Интерактивный пример justify-content:
1
2
3

Текущее значение: flex-start

Возможные значения:

  • flex-start (по умолчанию): элементы прижимаются к началу контейнера
  • flex-end: элементы прижимаются к концу контейнера
  • center: элементы центрируются по главной оси
  • space-between: равномерно распределяет пространство между элементами
  • space-around: равномерно распределяет пространство вокруг элементов
  • space-evenly: равномерно распределяет пространство между всеми элементами и краями контейнера

Свойство align-items

Свойство align-items определяет, как flex-элементы располагаются вдоль поперечной оси контейнера.

Возможные значения:

  • stretch (по умолчанию): растягивает элементы, чтобы заполнить контейнер
  • flex-start: элементы размещаются в начале поперечной оси
  • flex-end: элементы размещаются в конце поперечной оси
  • center: элементы центрируются по поперечной оси
  • baseline: элементы выравниваются по базовой линии текста

Свойства flex-элементов

Основные свойства, применяемые непосредственно к flex-элементам:

flex-grow

Определяет способность элемента увеличиваться в размере при наличии свободного места

order

Определяет порядок отображения элементов независимо от их расположения в DOM

flex-shrink

Определяет способность элемента уменьшаться в размерах при нехватке места
